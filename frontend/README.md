# FlowSight Frontend

n8n-style workflow graph viewer for FlowSight.

## Planned Architecture

- **Framework**: React + TypeScript
- **Graph Visualization**: React Flow or D3.js
- **State Management**: Zustand or React Query
- **Styling**: Tailwind CSS
